### Побудова грядок простої форми та саджання

1. Обрати інвентар - "Земля" (або "Рожеве скло")
2. Обрати інвентар рослини - квіти або інші саджанці
3. Обрати місце і натискаючи ПКМ рухатися вперед - буде саджатися
